### Styling OneBangs

#### Height

Names: 'height', 'h'

Accepts: One CSS size statement

```html
<div !height:10px></div> <!-- <div style="height: 10px;"></div> -->
<div !h:6em></div> <!-- <div style="height: 6em;"></div> -->
```

#### Width

Names: 'width', 'w'
